Homey Ping

Keep an eye on your network devices with Homey Ping.

Homey Ping lets you monitor IP addresses and hostnames and use their online or offline state in Homey Flows. It is ideal for checking whether devices such as NAS systems, printers, access points, media players, or other network equipment are reachable.

Features:
- Add a host or IP address manually
- Monitor availability at a custom interval
- Use Auto mode for ICMP with TCP fallback
- Use ICMP only or TCP only when needed
- Clear device status values: Online / Offline
- Manually trigger a check with Ping now
- Use Flow cards for:
  - Device came online
  - Device went offline
  - Is online
  - Is offline
  - Ping now

Homey Ping is designed to work reliably across different Homey environments. If ICMP is unavailable on your system, use Auto mode or TCP only.
