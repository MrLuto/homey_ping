Monitor IP addresses and hostnames on your local network and use their online or offline state in Homey Flows. This app is useful for devices such as NAS systems, printers, access points, media players, and other network equipment.

You can add devices manually and monitor them at a custom interval with configurable timeout. Probe mode can be set to Auto, ICMP only, or TCP only. Auto mode uses ICMP first and falls back to TCP when ICMP is not available.

Flow cards are available to trigger when a device comes online or goes offline, check current online or offline state, and run a manual check with Ping now.
